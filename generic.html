PRÉSENTATION

Projet

L'objectif de ce projet est de réaliser une application capable de générer une ville avec ses différentes structures.

structures

  • Mairie
  • Habitations
  • Bureaux
  • Magasins
  • Stations métro

sécurité

  • Commisariats

santé

  • Hopitaux

Vous pouvez voir ci-dessus que nous avons réalisé une légende pour la lecture de la carte générée indiquant le type de bâtiment pour chaque graphisme correspondant.

Vous pouvez donc voir sur l'image ci-dessus les différentes composantes de l'interface utilisateur GTK.

L'image ci-dessus illustre le passage d'une matrice d'int à une image SDL affichée.

Le fonctionnement de la dernière version de l'algorithme réutilise la structure de données établie précédemment pour les stations de metro, avec une nouvelle structure pour la gestion des arcs. Pour cette dernière version, on a supprimé la range de distance maximum entre deux stations ce qui fait que le programme s’adapte aussi bien sur une petite ville que sur une grande. On construit la liste de stations de métro, ainsi l'on sauvegarde les stations à traiter avec leurs coordonnées dans la matrice qui servira pour les labels des nœuds du graphe, gérer la notion de distance et donc la construction des lignes.
